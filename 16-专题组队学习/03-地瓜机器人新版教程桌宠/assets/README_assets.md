## 五、关于 OpenClaw 与 nanobot 资料来源的说明

`OpenClaw` 相关页面来自官方文档站，适合直接作为教程引用和归档。

`nanobot_club.html` 仅作为社区整理页面归档，便于保留当时的对比材料。涉及 `nanobot` 的版本、安装方式和能力边界时，实际写作应以官方仓库 `HKUDS/nanobot` 为主，而不是以该社区站点作为唯一依据。
