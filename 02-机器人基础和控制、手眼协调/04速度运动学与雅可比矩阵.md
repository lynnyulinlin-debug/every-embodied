#### 1. 简介：为什么它是“桥梁”？

我们在前几个文档中学习了**位置**（运动学）和**力/力矩**（动力学）。

**雅可比矩阵 (Jacobian Matrix, $J$)** 之所以被称为“桥梁”，是因为它在微芬层面连接了这两个领域：

1. **连接关节空间与笛卡尔空间**：它描述了关节速度 ($\dot{q}$) 如何映射为末端执行器的线速度和角速度 ($v, \omega$)。

   $$v = J(q) \dot{q}$$

2. **连接运动与力（虚功原理）**：这是动力学的入口。它描述了末端受到的力 ($F$) 如何转化为关节需要提供的力矩 ($\tau$)。

   $$\tau = J^T(q) F$$

如果没有雅可比矩阵，我们就无法根据末端想要的速度反推关节转速，也无法根据末端抓取的负载计算关节电机需要多大的力矩。

#### 2. 核心定义

对于一个 $n$ 自由度的机器人，其关节变量为 $q = [q_1, \dots, q_n]^T$。末端位置 $x$ 是 $q$ 的函数 $x = f(q)$。

雅可比矩阵定义为偏导数矩阵：

$$J(q) = \frac{\partial f}{\partial q} = \begin{bmatrix} \frac{\partial x}{\partial q_1} & \cdots & \frac{\partial x}{\partial q_n} \\ \vdots & \ddots & \vdots \\ \end{bmatrix}$$

#### 4. Python 代码验证：计算 2 连杆机械臂的雅可比

我们使用 `sympy` 符号推导来自动生成雅可比矩阵。



```Python
import sympy as sp

def calculate_jacobian_2link():
    # 1. 定义符号
    theta1, theta2 = sp.symbols('theta1 theta2')
    l1, l2 = sp.symbols('l1 l2')

    # 2. 正运动学方程 (末端 x, y 坐标)
    # x = l1*c1 + l2*c12
    # y = l1*s1 + l2*s12
    x = l1 * sp.cos(theta1) + l2 * sp.cos(theta1 + theta2)
    y = l1 * sp.sin(theta1) + l2 * sp.sin(theta1 + theta2)

    # 位置向量
    pos = sp.Matrix([x, y])
    
    # 关节变量向量
    q = sp.Matrix([theta1, theta2])

    # 3. 计算雅可比矩阵 J = d(pos)/d(q)
    J = pos.jacobian(q)

    print("--- 符号形式的雅可比矩阵 J ---")
    sp.pprint(J)

    # 4. 数值验证 (奇异点测试)
    # 当 theta2 = 0 时 (手臂完全伸直)，机械臂处于奇异形位，
    # 此时雅可比矩阵行列式应为 0 (失去一个自由度)
    J_func = sp.lambdify((theta1, theta2, l1, l2), J, 'numpy')
```
