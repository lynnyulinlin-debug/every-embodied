# 地瓜机器人新版教程桌宠

这组教程聚焦地瓜机器人新版桌宠方向，保留 OpenClaw / MagicBox 的部署、语音、按钮、灯效与控制脚本，不纳入不需要公开的场景编辑器与本地大包。

## 本专题包含什么

1. 开发环境准备
2. OpenClaw / MagicBox 接入
3. 语音、按钮、灯效、舵机控制
4. 桌宠交互的发布流程
5. 发布时要排除的本地下载物和缓存
